slimta.util¶

Package containing a variety of useful modules utilities that didn’t really belong anywhere else.

slimta.util.validate_tls(tls[, **overrides])¶

Given a dictionary that could be used as keyword arguments to ssl.wrap_socket, checks the existence of any certificate files.

Parameters:
  • tls (dict) – Dictionary of TLS settings as might be passed in to an Edge constructor.
  • overrides (keyword arguments) – May be used to override any of the elements of the tls dictionary.
Returns:

The new, validated tls dictionary.
Raises:

OSError
