slimta.relay.smtpÂ¶

Package implementing the ability to route messages to their next hop with the standard SMTP protocol. Moving messages from hop top hop with SMTP is the foundation of how email works. Picking the next hop is typically done with MX records in DNS, but often configurations will involve local static routing.
