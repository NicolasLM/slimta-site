slimta.util.pycompat¶

This module makes compatibility between Python 2 and Python 3 a little more convenient. It’s intended to avoid dependence on the six library.

slimta.util.pycompat.PY3 = False¶

True if the interpreter is Python 3.x, False otherwise.

slimta.util.pycompat.PY2 = True¶

True if the interpreter is Python 2.x, False otherwise.
